### Installation

1. Clone the repo
  ```sh
    gh repo clone Lysaker-git/Semester-Project-2_Noroff
  ```
2. Install NPM packages
  ```sh
    npm install
  ```
3. Run Build to enable SASS
  ```
  npm run build
  ```
